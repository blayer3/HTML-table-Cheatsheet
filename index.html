HTML5 Tables

Table Tags

Tag Name Description
<table> Table Defines a table in html.
<thead> Table Header Groups the header content of a table.
<tbody> Table Body Groups the body content of a table.
<tfoot> Table Footer Groups the footer content of a table.
<tr> Table Row Defines a table row.
<td> Table Data Defines a table cell.
<th> Table Heading Defines a table heading cell (used within thead).
<caption> Table Caption Defines a caption for the table.
<colgroup> Column Group Specifies a group of one or more columns in a table for formatting.
<col> Column Specifies column properties for each column within a colgroup element.

Table Attributes

Attribute Name Description
colspan Column Span Defines the number of columns that a cell spans.
rowspan Row Span Defines the number of rows that a cell spans.
